Aprendizaje supervisado parte 1 -- Clasificación¶

Para visualizar como funcionan los algoritmos de aprendizaje automtico, es mejor considerar datos de una o dos dimensiones, esto es datasets con solo una o dos caractersticas. Aunque, en la prctica los datasets tienen muchas ms caractersticas, es difcil representar datos de alta dimensionalidad en pantallas 2D.

Vamos a ilustrar ejemplos muy simples antes de comenzar con datasets del mundo real.

Primero, vamos a inspeccionar un problema de clasificacin binaria con dos dimensiones. Utilizaremos los datos sintticos que nos proporciona la funcin make_blobs.

In [2]:

La clasificacin es una tarea supervisada y, ya que estamos interesados en su rendimiento en datos no utilizados para entrenar, vamos a dividir los datos en dos partes:

  1. un conjunto de entrenamiento que el algoritmo de aprendizaje utiliza para ajustar los parmetros del modelo
  2. un conjunto de test para evaluar la capacidad de generalizacin del modelo

La funcin train_test_split del paquete model_selection hace justo esto por nosotros - la usaremos para generar una particin con un 75%//25% en entrenamiento y test, respectivamente.

In [4]:
from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(X, y,
                                                    test_size=0.25,
                                                    random_state=1234,
                                                    stratify=y)

Otro clasificador: K Nearest Neighbors¶

Otro clasificador popular y fcil de entender es el k Nearest Neighbors (kNN). Implementa una de las estrategias ms simples de aprendizaje (de hecho, en realidad no aprende): dado un nuevo ejemplo desconocido, buscar en la base de datos de referencia (entrenamiento) aquellos ejemplos que tengan caractersticas ms parecidas y asignarle la clase predominante.

La interfaz es exactamente la misma que para LogisticRegression.

from sklearn.neighbors import KNeighborsClassifier
